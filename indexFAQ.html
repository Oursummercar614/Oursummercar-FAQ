Game Development FAQ

Release date?

When it's ready, however you can check download page and check the reverse-open builds over time.

When did development start?

March 2025

Will find support phase copy of the game?

No. Most requires original steam copy of the game to work.

How many players will the multiplier support?

Currently multiplier supports up to a players in a single session. Plans include making GTA Online-style multiplayer (with online support) and possible future support for more players.

Will there be dedicated servers?

Currently no. The game is using peer-to-peer networking. One player is a host that connects via Steam to other players. The host player's save is being used.

What about custom images?

The game will download custom images that the "host" has. When you join a game, your client will download and load other players' images before streaming begins.

Who is making this game?

The game is being developed by Epic, Citrix, and Dark Kostas, with contributions from other community members.

Need the Red Leader?

No custom functions required. Multiplayer works with different games (like ETS 2 and GTA-SA). Just launch the game using the multiplayer mod and connect through your desktop.

Can I use mods with multiplayer?

Mods won't be blocked by the multiplayer mod, but synchronization isn't planned for initial release. A multiplayer API will be released for mod creators to properly synchronize their creations.

What happens when one of the players disconnects?

If host disconnects: the game ends. If joining player disconnects: they can attempt to reconnect.

How is time-skipping handled?

Currently synchronizes time after time-skipping activities. Considering either disabling time-skipping or making it "synchronized" where all players must perform the same time-skip.
